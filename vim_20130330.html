使用vimwiki记笔记

为什么选择vimwiki？

基于vim

输出为html

简洁但不简陋

http://xbeta.info/vimwiki.htm Pkm工具：Vimwiki

http://www.vim.org/scripts/script.php?script_id=2226 vimwiki : Personal Wiki for Vim

windowsxp

下载vimwiki-2-0-1.vba.gz：http://www.vim.org/scripts/script.php?script_id=2226

解压至任意位置

用vim打开vimwiki-2-0-1.vba

执行命令：so %

_vimrc中添加

set nocompatible

filetype plugin on

syntax on

" vimwiki

" path:存放Vimwiki文件的路径

" path_html:从Vimwiki转换为html网页时的保存路径

" html_header:Vimwiki转换网页时使用的网页模板的路径

let g:vimwiki_use_mouse = 1

let g:vimwiki_list = [{'path': 'D:/vimwiki/',

\ 'path_html': 'D:/vimwiki/html/',

\ 'html_header': 'D:/vimwiki/template/header.tpl',}]

nmap <F5> :Vimwiki2HTML<CR>

nmap <F6> :VimwikiAll2HTML<CR>

重启vim

在Vim的Normal模式下，连续按下 \ww 打开wiki首页

<F6>生成html

查看vimwiki帮助信息的命令：help vimwiki

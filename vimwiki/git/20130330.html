使用github托管个人主页

下载git http://code.google.com/p/msysgit/downloads/list?q=full+installer+official+git

安装git，一路选择默认配置到底

注册 github : https://github.com/

打开安装好的git bash，依次输入：

git config --global user.name "your username"

git config --global user.email "your email"

用户名和邮箱要与注册github时使用的一致。

创建SSH Public Keys，输入：

ssh-keygen -t rsa -C "your email"

回车后，会提示输入保存key的文件名，一般用id_rsa：

Generating public/private rsa key pair.

Enter file in which to save the key (/c/Users/Tekkub/.ssh/id_rsa):

输入完会要求设置密码。

Enter passphrase (empty for no passphrase):

Enter same passphrase again:

完成后提示id_rsa.pub文件保存的位置。

打开id_rsa.pub，将其中的内容复制到 github 网站的 Account Settings 的 SSH Keys 中。

以后就不需要重复上面的操作了。

在 github 中创建名为 用户名.github.com 的项目，并使用自动功能创建page，十几分钟后就可以通过 用户名.github.com 访问个人主页了。

clone 上面的项目至本地。

修改上传网页即可。
